## mozartforschung.de

<img src="src/images/mozartforschung.jpg" alt="domain image" style="width: 600px;"/>

[Website](https://mozartforschung.de) mit Beiträgen zur Mozartforschung von Ulrich Kaiser.

Alle Forschungsbeiträge stehen unter der jeweils angegebenen Creative-Commons-Lizenz. Die folgende Buchpublikation war bis 2013 im Bärenreiter-Verlag erhältlich und wird hier bereitgestellt (Open Access):

Ulrich Kaiser, Die Notenbücher der Mozarts als Grundlage der Analyse von W. A. Mozarts Kompositionen 1761–1767, Kassel 2007, ²2013 (mit Korrekturen im Selbstverlag).
